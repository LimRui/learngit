git命令
1.进入git
cd xxxx(文件盘)
git init
2.添加文件
git add xxx(文件名全拼)
git commit -m "wrote a xxx file"

3.推送文件到远程服务器上
git remote add origin https://github.com/LimRui/learngit.git //我的服务器地址
git push origin master

4.修改文件
(1)查看仓库当前状态
	git status
(2)查看修改什么内容
    git diff xxx(要查看的文件)

5.分支
(1)创建分支
git checkout -b dev
(2)查看分支
git branch
(3)切换分支
git checkout master
(4)合并dev分支到master上
git merge dev
(5)删除分支
git branch -d dev
